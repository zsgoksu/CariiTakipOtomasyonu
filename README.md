# CariiTakipOtomasyonu
<p>Kullanıcı cari hesapları tutabilir, ürünleri kayıt altına alabilir ve bu ürünlerin stok takibini yapabilir. İşletmeye ait hesap hareketlerini raporlanması, gider ve diğer gelirlerin kayıt altına alınması özellikleri de projede yer almaktadır. Ayrıca kullanıcıya satış, alacak ve borç için bir raporlama sunulmaktadır.  
  
Kullanılan Teknolojiler : Java Swing, SQLite 


</p>
